# Contributing

Thanks for improving Aegis Atlas.

## Development setup

1. Fork or clone the repository.
2. Create a focused branch from `main`.
3. Install the locked dependencies with `npm ci`.
4. Start the app with `npm run dev`.

## Quality checks

Run the complete local verification before opening a pull request:

```bash
npm run check
```

Keep changes focused, add or update tests for behavior changes, and update documentation when user-facing behavior changes.

## Pull requests

Explain the problem, the chosen approach, and validation performed. Link related issues and include screenshots for visual changes. Do not include secrets, generated build output, or unrelated formatting changes.
